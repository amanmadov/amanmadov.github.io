# Certification Logos

Add your certification logos here. Each logo should be:

- PNG or SVG format
- Square dimensions (recommended: 200x200px minimum)
- Transparent background preferred
- Named as referenced in the code

## Current certifications that need logos:

1. aws-logo.png - AWS Logo
2. kubernetes-logo.png - Kubernetes Logo
3. terraform-logo.png - HashiCorp Terraform Logo
4. salesforce-logo.png - Salesforce Logo
5. uipath-logo.png - UiPath Logo
6. azure-logo.png - Microsoft Azure Logo

You can download official logos from:

- AWS: https://aws.amazon.com/architecture/icons/
- Kubernetes: https://kubernetes.io/community/
- HashiCorp: https://www.hashicorp.com/brand
- Salesforce:
  https://developer.salesforce.com/docs/atlas.en-us.brand.meta/brand/
- UiPath: https://www.uipath.com/brand
- Microsoft Azure: https://azure.microsoft.com/en-us/resources/brand-assets/

Or you can remove the `logo` property from certifications in
`src/pages/index.astro` if you don't want to display logos.
